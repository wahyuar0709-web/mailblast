Selamat datang di MailBlast!

          

            Aplikasi ini membantu kamu kirim lamaran kerja ke ratusan perusahaan secara otomatis.


            Setup hanya butuh 2 menit. Ikuti langkah-langkah berikut.
          

        

        

          
Yang akan kamu setup:

          

            
✅ Koneksi ke Google Sheets (tempat data tersimpan)

            
⬜ Buat tabel data otomatis di Spreadsheet

            
⬜ Nama & email pengirim

          

        

        
      


      
      

        
Hubungkan ke Google Sheets

        

          MailBlast menyimpan data di Google Sheets milikmu. Kamu perlu URL Google Apps Script yang sudah di-deploy.
        


        

          
📋 Cara mendapatkan URL:

          
1. Buka script.google.com

          
2. Buat project baru → paste kode MailBlast.gs

          
3. Klik Deploy → New deployment

          
4. Pilih type: Web app → Execute as: Me

          
5. Copy URL yang muncul → paste di bawah

        


        

          
          
        


        


        

          
          
          
        

      


      
      

        
Buat Tabel di Spreadsheet

        

          MailBlast perlu membuat beberapa sheet di Spreadsheet kamu. Klik tombol di bawah — prosesnya otomatis.
        


        

          
Sheet yang akan dibuat:

          

            
📋 PT_List — daftar perusahaan

            
📧 Template — template email

            
📎 Lampiran — daftar file lampiran

            
📊 Log — riwayat pengiriman
